## Contents

The repository includes the following Jupyter notebooks:

### `knn.ipynb`
Implementation and experiments for the **k-Nearest Neighbor (kNN)** classifier on CIFAR-10.

### `softmax.ipynb`
Implementation of a **Softmax classifier**, including vectorized loss and gradient, and hyperparameter tuning.

### `two_layer_net.ipynb`
Implementation of a **two-layer fully connected neural network** trained with backpropagation.

### `FullyConnectedNets.ipynb`
Implementation and training of **deeper fully connected networks**, including techniques like regularization and optimization strategies.
